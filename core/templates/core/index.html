Teal CRM

A lightweight, multi-tenant Django CRM with team workspaces, plan-based limits, lead-to-client conversion, and a clean responsive dashboard — ideal for learning or extending a small sales workflow.

Sign up Log in View dashboard

What it does

Manage leads and clients inside team workspaces. Each team can be assigned a plan that limits how many leads and clients can be created — useful for experimenting with multi-tenant and billing-related logic in a compact codebase.

Leads & Clients

Create, edit, convert leads to clients and track statuses and priorities.

Teams & Plans

Per-team limits (plans) control creation counts and act as a simple tenant boundary.

Responsive UI

Tailwind-based, minimal and easy to customize for learning or demos.

Admin-ready

Built-in admin to manage teams and plans for quick testing or seeding data.

Quick start

  • Clone, create a Python venv, install requirements and run migrations.
  • Create a plan and ensure each user/team has a plan before creating many leads/clients.
  • Use the admin for fast data seeding and testing.
